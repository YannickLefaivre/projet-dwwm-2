# Project Delta

Un site d'orientation scholaire.

## Guide d'installation

### Étape 1 : Cloner le repo sur votre ordinateur

```
git clone 
```

### Étape 2 : Installer les dépendences du projet 

```
npm install
```

### Étape 3 : Démarrer l'application

```
npm start
```
